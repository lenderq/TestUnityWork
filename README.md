# Lootbox Slot Test Task

Тестовое задание выполнено на базе проекта с AxGridTools в Unity 6.

## Реализовано
- Мини-слот с 3 изображениями в вертикальный ряд.
- Бесконечный скролл вниз с переиспользованием элементов.
- Плавный разгон после нажатия Start.
- Кнопка Stop становится доступной через 3 секунды после старта.
- Плавная остановка с выравниванием центрального элемента по центру окна.
- Визуальное оформление: фон, рамка, иконки.
- Система частиц при остановке.
- Блокировка кнопок через `UIButtonDataBind`.

## Архитектура
Логика разделена на слои:
- **FSM** — состояния и переходы (`Idle`, `Spinning`, `CanStop`, `Stopping`)
- **View** — прокрутка, торможение, выравнивание, VFX
- **UI/Binding** — отправка сигналов из кнопок и управление доступностью через модель

Использован поток:
`UI -> FSM -> View -> FSM`

## Основные файлы
- `SlotFsmBootstrapper` — инициализация и обновление FSM
- `SlotStateBase` + состояния — логика слота
- `SlotReelView` — бесконечная прокрутка и остановка
- `SlotStopVfx` — частицы при остановке
- `SlotRootBootstrapper` — синхронизация модели
- `UIButtonDataBind` — работа кнопок через AxGrid binding

## Примечание
В проекте оставлены debug-логи для наглядной проверки переходов FSM и событий.
